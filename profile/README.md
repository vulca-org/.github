# Vulca Systems

Vulca Systems builds evidence-governed, local-first agent products and
infrastructure. Our repositories separate product execution, human review,
research evidence, integrations, and organization governance so each project
has one clear responsibility.

## What we build

- **Organization Control Plane** — public policy, repository authority,
  migration, and release gates.
- **Visual Systems** — visual intent, controlled execution, semantic editing,
  and review workspaces.
- **Writing Systems** — evidence-controlled academic writing, citation,
  revision, and release workflows.
- **Agent Infrastructure** — context, independent review, multimodal gates, and
  limit-aware operating discipline.
- **Research and Evaluation** — cultural benchmarks, evaluation frameworks,
  data, and bounded research artifacts.
- **Integrations, Internal, and Archive** — provider adapters, restricted
  operations, and read-only historical assets.
